## 💾 Installation

#### 👇 Hit in your terminal for setup

```bash
pnpm i
```

## 🥑 Usage / Get Started

```bash
# Compiles & hot-reloads for development
pnpm dev

# Vite preview
pnpm preview

# Build & minifies for production
pnpm build

# Lints & fixes files
pnpm lint:scripts

pnpm lint:styles

pnpm lint:markup

# Prettier & format files
pnpm format
```

#### Goto application

App running at on [localhost](http://localhost:5173)
